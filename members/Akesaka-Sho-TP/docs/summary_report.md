## 2. 設計で考えたこと（15秒）

LEDの色ごとに音階を割り当てることで、視覚的にも音の違いが分かるように工夫しました。

## 3. できたこと・できなかったこと（30秒）

<!-- 正直に書いてください。「できなかった」も立派な成果です -->

**動いたもの：**
- パッシブブザーで曲が流れた
- LEDを音階ごとに色が光るようになった

**動かなかった・間に合わなかったもの：**
- LCDディスプレイの配線が複雑で繋げてみたが、つながらない
- なぜ動かなかったか（わかる範囲で）：あきらかに配線の位置があってなかったから。

## 4. 一番苦労したこと、どう乗り越えたか（30秒）

<!-- ここが発表の山場です。1つだけに絞って、ストーリーで書いてください -->

**何が起きたか：**
- 音がなっていたのに音が流れなくった
- LEDが一つ光ったが他のLEDが光らない、光ったと思ったら全部のLEDのライトが光らなくなった

**どう対処したか：**
両方とも配線の位置に問題があった。


**そこから何がわかったか：**
配線が少し違うだけで全く動かなくなることを実感しました。正しい回路図を見ながら一つずつ確認することの大切さを学びました。
