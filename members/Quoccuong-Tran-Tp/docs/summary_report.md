## 1. 何を作ったか（30秒）

<!-- ひと言で伝わるように書いてください。「○○を使って、△△するガジェットを作りました。」 -->

**ガジェット名：**忘れ物防止システム

**ひと言で説明：**超音波センサーを使って人の離席を検知し、確認されない場合に警告するガジェットを作りました。

**使った部品：**
- 超音波センサー（HC-SR04）
- Arduino（マイコン）
- LED・ブザー・ボタン

<!-- 正直に書いてください。「できなかった」も立派な成果です -->

**動いたもの：**
- 人の在席検知と離席判定

- ボタンによる停止機能

**動かなかった・間に合わなかったもの：**
